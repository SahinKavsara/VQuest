1. Kişisel Analiz Başlatma (AI)

API Metodu: POST

Açıklama:
Kullanıcının oyun içi verilerinin yapay zekâya gönderilerek kişisel performans ve gelişim analizinin başlatılması.

⸻

2. Bildirim Gönderme

API Metodu: POST

Açıklama:
(Admin) Sistem yöneticisinin tüm kullanıcılara anlık bir sistem duyurusu veya bilgilendirme mesajı göndermesi.

⸻

3. Yapay Zekâ Komutu Güncelleme

API Metodu: PUT

Açıklama:
(Admin) Yapay zekânın analiz üretirken arka planda kullandığı komutun (prompt) güncellenmesi.

⸻

4. Bildirim Okundu İşaretleme

API Metodu: PUT

Açıklama:
Kullanıcının kendisine gönderilen bir sistem bildirimini “okundu” olarak işaretlemesi.

⸻

5. Analiz Sonucu Görüntüleme (AI)

API Metodu: GET

Açıklama:
Yapay zekâ tarafından üretilen kişisel gelişim ve performans analiz raporunun kullanıcıya getirilmesi.

⸻

6. Bildirim Görüntüleme

API Metodu: GET

Açıklama:
Kullanıcının kendisine gelen tüm uyarı ve bilgilendirme bildirimlerinin listelenmesi.

⸻

7. Eski Analizleri Silme (AI)

API Metodu: DELETE

Açıklama:
Kullanıcının geçerliliğini yitirmiş veya artık ihtiyaç duymadığı eski yapay zekâ analiz raporlarını silmesi.

⸻

8. Bildirim Silme

API Metodu: DELETE

Açıklama:
Kullanıcının okuduğu veya artık görmek istemediği bildirimleri kendi bildirim kutusundan tamamen kaldırması.
